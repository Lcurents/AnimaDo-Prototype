# AnimaDo — Prototype

A small web prototype for simple animations and timed interactions.

## Features

- Minimal UI to start/reset an animation flow.
- Example controls: [`play`](script.js), [`reset`](script.js).
- Open the demo: [index.html](index.html)

## Usage

1. Open [index.html](index.html) in your browser.
2. Click "Start" (calls [`play`](script.js)) to begin.
3. Click "Reset" (calls [`reset`](script.js)) to reset.

## Files

- [index.html](index.html)
- [script.js](script.js)

# AnimaDo — Prototipe

Sebuah prototipe web sederhana untuk animasi dan interaksi berbasis waktu.

## Fitur

- UI minimal untuk memulai/mengatur ulang alur animasi.
- Kontrol contoh: [`play`](script.js), [`reset`](script.js).
- Buka demo: [index.html](index.html)

## Cara Pakai

1. Buka [index.html](index.html) di peramban Anda.
2. Klik "Start" (memanggil [`play`](script.js)) untuk mulai.
3. Klik "Reset" (memanggil [`reset`](script.js)) untuk mengatur ulang.
